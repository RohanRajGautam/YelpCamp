## YelpCamp :tent:
YelpCamp is a full stack app built using NodeJs, Express & MongoDB as a part of colt steele's web developer bootcamp on udemy.

### Used the following:
* HTML/Bootstrap
* CSS
* JavaScript
* Node.js
* mongoDB
* npm packages: 
    * body-parser
    * express, express-session
    * ejs
    * mongoose
    * passport, passport-local, passport-local-mongoose
    * method-override
    * connect-flash
